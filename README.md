# Form data manipulation with React and Node.js

# Installation

`git clone https://github.com/forhad-h/react-node-form.git`  
`cd react-node-form`  
`yarn start`

### Take another terminal/cmd

`node server.js`

## Used Module

- `axios`
- `cors`
- `express`
- `multer`
- `node-sass`
- `react`
- `react-dom`
- `react-scripts`
- `react-toastify`
- `reactstrap`
